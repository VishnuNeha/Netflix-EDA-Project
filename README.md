# Netflix Exploratory Data Analysis (EDA)

## Overview
This project performs Exploratory Data Analysis (EDA) on the Netflix Titles dataset to identify trends, patterns, and insights related to Netflix's content library.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Analysis Performed
- Data Cleaning
- Missing Value Analysis
- Statistical Summary
- Content Type Analysis
- Ratings Analysis
- Genre Analysis
- Country Analysis
- Trend Analysis
- Correlation Analysis

## Key Findings
- Movies make up around 70% of Netflix content.
- The United States is the leading content producer.
- TV-MA is the most common rating.
- International Movies are the most popular genre category.
- Netflix content growth increased significantly until 2019.
